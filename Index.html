Bish Bosh Ver. 2

Bish-Bosh lists all numbers between 1 and counting value, but some exceptions: If the number is evenly divisible by Bish, ​`Bish' ​is listed instead of the number.

If the number is evenly divisible by Bosh, 'Bosh' is listed instead of the number.

If the number is evenly divisible by both Bish and Bosh, “​Bish-Bosh​” is written instead of the number.
